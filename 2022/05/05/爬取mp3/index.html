一次使用爬虫解决个人mp3曲库问题实践

记一次升级pip : python -m pip install --upgrade pip，cmd 中的一长串（C:\Users\Frank\AppData\Local\Programs\Python\Python310\python.exe -m pip install --upgrade pip）是python文件的运行路径，如果把启动添加进环境变量，就用python代替

第一版正文：

实现功能：

比较粗糙，只有下载而已，而且键盘输入比较麻烦、没有批量高并发下载，后面再加功能

使用工具及第三方包：

  1. requests 很方便的网络请求工具
  2. pip install requests
  3. 文档比较好用，说人话，适合正常人阅读：https://docs.python-requests.org/zh_CN/latest/user/quickstart.html#id5
  4. python 内置的文件管理
  5. postman 用于测网络请求
  6. web 自带的调试工具 （浏览器按 F12）

然后记一下比较重要的环节

  1. 如果 post 或是 get 的返回数据是键值对，可以做 json 处理，res.json()。转化成键值对格式方便访问，如果返回的是 html文档，那就只能用 BeautifulSoup 截取字符串来获取想要的url

  2. 流式数据下载及存储：

    'wb' 这个参数意味着如果不存在这个文件就创建，如果存在就重置，可以去看文档，参数有很多

    路径用 ./ 意味着用的是命令行当前的相对路径，不是工具文件位置的相对路径

  3. 系统命令行输入，使用 sys 模块，后面再完善（getopt）
