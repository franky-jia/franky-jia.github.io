  1. 搭建博客

    1. 环境
      1. git
      2. node.js
        1. 使用npm进行包管理
          1. -g 全局安装，包 位置C:\Users\Frank\AppData\Roaming\npm
    2. 生成初始项目

      1. npm install hexo-cli -g 安装hexo脚手架

      2. hexo init blog 生成blog项目

      3. cd <blog>
npm install

hexo generate	//生成新的静态页面文件
hexo server		//启动本地服务，可以自己改端口
