HTML CSS 总结

  • HTML语义化
    • 增加代码可读性
    • 利于SEO
  • 块状元素 & 内联元素
    • 块状：block/table，表现为独占一行
    • 内联：inline/inline-block，表现为在一行内
  • 盒模型尺寸计算
    • offsetWidth = width+padding+border
    • box-sizing: border-box
  • margin 纵向重叠问题
    • 纵向相邻的两个元素的margin-top与margin-bottom会相互重叠
  • margin负值（没什么特别的，就margin为负数，外部计算位置的时候要加上负值）
  • BFC （）
    • 一块独立渲染区域，内部元素渲染不受外部因素影响
    • BFC形成条件
    • BFC常见应用，清除浮动
  • 圣杯布局 双飞翼布局（卵用没有）
  • absolute 和 relative（有用，但过时了，用手找px，脑子有问题）
  • 居中对齐方式：
    • inline 元素：水平方向 text-align: center 垂直方向 line-height 等于 height
    • block 元素：margin: auto
  • rem 布局方式（先不看）
  • CSS动画（先不看）

JS基础

变量类型与计算

  • typeof能判断那些类型：

    • 除了引用类型和null都能判断，引用类型和null判断为object

    • 猜测null本身其实是个对象，js中的null指向的是同一个东西

  • === 与 ==

    • === 是全等，要求数据类型也要等同
    • == 不严格要求数据类型

  • 值类型 & 引用类型

    • 值类型赋值

    • 引用类型赋值

  • 手写深拷贝

    • 遍历 + 递归 即可 非常简单
    • 注意，遍历使用 for( key in obj) 挺好用的，对于数组拿到的key是0，1，2这种
原型链

  • 使用 class 实现继承，比掌握原型链简单多了，记住几个关键字和用法就完了

    • class
    • extends
    • constructor 声明属性
    • super 继承的时候选择哪些属性继承（事实上不带super也会把属性继承下来，但是在new实例的时候会无法对该属性赋值，该属性的值为undifined）
作用域 与 闭包

  • 作用域与自由变量

    • 全局作用域，函数作用域，块级作用域（for，if 包裹的区域）
    • 自由变量，当前作用域没有定义但被使用的变量，会一层层往外层作用域查找

  • 闭包

    • 自由变量的查找是在函数定义的地方，而不是执行的地方
    • 对于带有返回函数的函数来说，返回函数的外层作用域就是函数

  • this

    • this是在执行的时候确定的，不是在定义的时候确定的

    • 使用场景

      • 作为普通函数
      • 使用 call apply bind
      • 作为对象方法被调用
      • 在 class 方法中被调用
      • 箭头函数

    • apply，bind，call 手写（感觉有点蠢，用apply写个bind）

      • Array.prototype.slice.call(arguments) 把类数组arguments转成数组
      • 同时 self = this 这步存储 this 也是种常用做法，必须掌握
      • 这个东西的基础是函数式编程，了解函数式编程对写这个代码很有帮助，其实就是返回一个能执行apply的函数

    • 闭包隐藏数据
